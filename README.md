vidplay
=======

Python wrapper class for omxplayer Raspberry Pi video player

Can run from the command line test program

> usage: python test_videoplayer.py <arg>
> where <arg> is:
> 
> 1 -- play video 1
> 2 -- play video 2
> s -- stop video
> p -- pause video
> t -- test if there is a video playing
> x -- stop videoplayer and exit
